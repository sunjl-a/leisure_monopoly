# 城市大富翁

一个纯前端实现的本地同屏大富翁小游戏。玩家可以在中国城市主题棋盘上购买城市、建设房屋、收取租金、参与拍卖，并通过资产经营让对手破产，最终成为最后的胜者。

## 项目特点

- 支持 2-4 名玩家同屏游玩。
- 支持真人玩家和电脑玩家混合对局。
- 采用中国城市、交通枢纽和公共事业主题棋盘。
- 包含购买地块、收取租金、发起拍卖、抵押、赎回和建房等玩法。
- 内置机会卡和命运卡事件。
- 包含监狱、税费、免费停车、破产和最终排名结算。
- 使用 Web Audio API 生成简单音效。
- 无需构建工具或后端服务，直接用浏览器打开即可运行。

## 技术栈

- HTML
- CSS
- 原生 JavaScript

## 快速开始

直接在浏览器中打开 `index.html`：

```bash
open index.html
```

如果希望通过本地服务器访问，也可以在项目目录启动一个静态服务：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 玩法概览

1. 在开始界面选择 2-4 名玩家，并设置每位玩家为真人玩家或电脑玩家。
2. 每位玩家初始获得 ¥1500。
3. 玩家轮流掷骰并沿棋盘移动。
4. 到达无人拥有的城市、交通枢纽或公共事业时，可以购买或发起拍卖。
5. 到达其他玩家拥有且未抵押的地块时，需要支付租金。
6. 玩家可以在自己未抵押的城市地块上建房，提高租金收益；同组已拥有城市需要均衡建房。
7. 现金不足时，系统会尝试通过抵押资产处理债务；仍无法偿还则破产出局。
8. 当只剩一名未破产玩家时，游戏结束并展示最终排名。

## 主要规则

- 经过起点可获得 ¥200。
- 保释费用为 ¥100。
- 每个城市最多建造 4 栋房屋。
- 每位玩家每回合最多建造 1 次房屋。
- 建房必须在玩家已拥有且未抵押的城市上进行。
- 同一颜色组内，玩家已拥有且未抵押的城市需要均衡建房，不能先把房屋更多的城市继续升级。
- 抵押地块会获得地块价格一半的现金，但抵押期间不会收取租金。
- 赎回抵押地块需要支付地块价格的 55%。
- 拥有同一颜色组的全部未抵押城市时，未建房城市的基础租金翻倍。
- 交通枢纽租金会随拥有数量提升。
- 公共事业租金根据本回合骰点数计算。

## 开发说明

项目没有外部依赖，所有玩法逻辑都集中在 `game.js` 中。棋盘数据、卡牌事件、玩家状态、回合流程、资产管理和 UI 渲染都由原生 JavaScript 管理。

如果要调整玩法，可以优先查看以下位置：

- `createBoard()`：棋盘地块配置。
- `createCards()`：机会卡和命运卡配置。
- `calculateRent()`：租金计算规则。
- `buildHouseBlockReason()`：建房限制规则。
- `renderBoard()`、`renderPlayers()`、`renderActions()`：主要界面渲染逻辑。
